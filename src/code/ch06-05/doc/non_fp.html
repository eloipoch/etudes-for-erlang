generate_teeth/2

generate_teeth(TeethPresent::string(), ProbabilityToBeGood::float()) -> [[integer()]]

Generate a list of lists, six numbers per tooth, giving random pocket depths. Takes a string where T="there's a tooth there" and F="no tooth"), and a float giving probability that a tooth is good.

generate_tooth/1

generate_tooth(ProbabilityToBeGood::float()) -> [integer()]

Generates a list of six numbers for a single tooth. Choose a random number between 0 and 1. If that number is less than the probability of a good tooth, it sets the "base depth" to 2, otherwise it sets the base depth to 3.
